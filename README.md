# Doctorly - Client Facing

This app template is for a Doctorly - Doctors Virtual Assistant Application - Client Side.

### Pages

#### Non-Protected pages

- Home Page
- Login Page
    - Enter Username
    - Enter OTP
- Forgot Password
    - Enter Username
    - Enter OTP
- Create Account

#### Protected pages

- Doctors list
- View Doctor
- Book an Appointment
- View Your Appointments
- View Appointment
- Settings

#### Components

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
